### Launching
***
##### Requirements:
* Ruby version _>= 2.3.0_
* Rails version _5.1.4_

##### Commands
Clone the repository to your local machine and run following commands:

```
$ bundle
```

```
$ bundle exec rake db:migrate
```

```
rails s
```

Open a `localhost:3000` page in your browser.
